# Preserve Edit Tool Line Endings Without Rewriting Bytes

**Source:** prompt.md item 40

**Priority:** P0

**Effort:** M

**Problem:** The edit tool normalizes CRLF and bare CR to LF, then restores CRLF by replacing every LF. This can rewrite bytes unrelated to the intended edit, including bare CR characters.

**Files involved:** `src/clients/tools/edit.rs`

**Fix:** Preserve original bytes outside edited ranges or avoid normalizing bare CR. Add tests proving unrelated CR and CRLF content remains unchanged.

**Status:** Completed
