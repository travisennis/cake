# Debug ast-grep exit 255 under Bash sandbox

Created: 2026-05-20

## Summary

Investigate and fix a suspected cake sandbox issue where commands executed
through the Bash tool cause ast-grep's `sg scan` to fail with exit code `255`
and little or no diagnostic output.

This was observed while running cake against `/Users/travisennis/Projects/acai-ts`.
The acai workflow uses `npm run check`, which runs `npm run lint:fix`, which
starts with `npm run lint:ast-grep`. That script runs `sg scan`. In cake agent
sessions, `sg scan` appeared to exit `255` silently, preventing the workflow
from reaching Biome complexity diagnostics.

The working hypothesis is that cake's Bash sandbox blocks something ast-grep's
binary needs: executable access, dynamic library access, config/rule traversal,
cache/temp access, symlink resolution, or another filesystem read/write path.

## Resolution

Completed: 2026-05-21

The current `sg scan` failure could not be reproduced and is not caused by
cake's Bash sandbox in the checked environment.

Reproduction from `/Users/travisennis/Projects/acai-ts`:

- Direct shell: `sg scan; printf '\nDIRECT_EXIT:%s\n' "$?"` exited `0` with
  no `sg scan` stdout/stderr.
- cake Bash tool, default sandbox:
  `sg scan; printf "\nCAKE_TOOL_EXIT:%s\n" "$?"` produced
  `CAKE_TOOL_EXIT:0` and the Bash tool footer `[exit:0 | 73ms]`.
- cake Bash tool, `CAKE_SANDBOX=off`:
  `sg scan; printf "\nCAKE_TOOL_EXIT:%s\n" "$?"` produced
  `CAKE_TOOL_EXIT:0` and the Bash tool footer `[exit:0 | 36ms]`.

The cake-generated macOS Seatbelt profile for the sandboxed run already grants
the paths ast-grep needs for this project:

- Read-write access to `/Users/travisennis/Projects/acai-ts`.
- Read-only/execute access to `/opt/homebrew`, where `sg` resolves on this
  machine.
- Read-write temp-directory access and `file-lock`, which cover the common
  runtime/cache operations suspected in the original report.

Because the sandboxed and unsandboxed cake runs both now succeed, there is no
profile rule to add for ast-grep. No cake code change or regression test was
added.
