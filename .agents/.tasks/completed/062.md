# Store Timestamps as DateTime Values Internally

**Source:** prompt.md item 65

**Priority:** P1

**Effort:** L

**ExecPlan:** `.agents/exec-plans/completed/store-timestamps-as-datetime.md`

**Problem:** Timestamps are generated as RFC3339 strings throughout the run path and stored as `Option<String>` in conversation/session data.

**Files involved:** `src/clients/types.rs`, `src/config/session.rs`, backend modules

**Fix:** Store `chrono::DateTime<Utc>` internally and serialize to RFC3339 at I/O boundaries. Handle old session files with defaults or compatibility deserializers.

**Status:** Completed
