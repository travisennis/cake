# Remove Bypass-Prone Bash Data Context Skips

**Source:** prompt.md items 31 and 75

**Priority:** P0

**Effort:** M

**Problem:** `is_data_context` skips all safety checks for segments starting with commands such as `echo`, `tee`, `cat >`, `git commit`, and `git tag`. Command substitutions inside those segments can still execute destructive operations.

**Files involved:** `src/clients/tools/bash_safety.rs`

**Fix:** Stop blanket-skipping segments based on command prefixes. Either inspect command substitutions or remove these skips and narrow exceptions to parsed literal arguments.

**Status:** Completed
