# Add Task Chaining with --then Flag

**Source:** product-ideas.md — Task Chaining

**Problem:** Cannot easily do "run this, then run that on the result" in a single invocation.

**Idea:** A `--then "follow-up prompt"` flag that automatically continues with a second prompt after the first completes. Alternatively, a simple task-file format (newline-separated prompts) that runs sequentially with `--task-file steps.txt`.

**Status:** Cancelled
