# Support --prompt-file for Reading Prompts from Files

**Source:** product-ideas.md — Prompt Files

**Problem:** Heredocs work but are clunky for complex, reusable prompts.

**Idea:** Support `cake --prompt-file task.md` to read the prompt from a file. Could also auto-detect `.cake/prompt.md` as a default prompt when no argument is given.

**Status:** Blocked
