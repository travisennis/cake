# Add Template Prompts from .cake/prompts/

**Source:** product-ideas.md — Template Prompts

**Problem:** Repeated tasks (code review, changelog, explain) require manual re-typing.

**Idea:** `.cake/prompts/review.md` templates invokable as `cake --template review`. Templates could include placeholder variables (`{{file}}`, `{{diff}}`) that get substituted from stdin or flags.
