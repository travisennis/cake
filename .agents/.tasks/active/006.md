# Add Lightweight Spinner/Status Line on stderr by Default

**Source:** product-ideas.md — Progress Feedback

**Problem:** `--verbose` is all-or-nothing; default mode is silent until task completes.

**Idea:** A lightweight spinner/status line on stderr by default (e.g., `⠋ Bash: ls -la...`) that does not require `--verbose`. `--verbose` remains for full detail; `--quiet` suppresses everything.
