# Add --add-file for Injecting Single Files into Context

**Source:** product-ideas.md — Context Injection

**Problem:** `--add-dir` exists for directories but there is no `--add-file` for injecting a single file's content into the prompt context.

**Idea:** `--add-file path` to inject a file's content directly into the user prompt. Useful for providing a spec, a test file, or a reference implementation without cat/heredoc gymnastics.

**Status:** Blocked
