# Add CLI Introspection Subcommands

**Source:** product-ideas.md — Discoverability

**Problem:** No way to list sessions or configured models from the CLI.

**Idea:** Add subcommands:
```bash
cake sessions list          # Show date, first prompt line, UUID for current directory
cake sessions search "auth" # Full-text search across session history
cake models list            # Show configured models from settings.toml
```
