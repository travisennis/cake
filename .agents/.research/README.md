# Research Artifacts

This directory stores research artifacts for cake. For the full workflow, read `.agents/RESEARCH.md`.

## Directory Layout

- `inbox/` contains raw ideas, pasted notes, and thin captures that have not been triaged.
- `investigations/` contains cake-specific findings from debugging, profiling, code reading, session analysis, or behavior checks.
- `sources/` contains notes from external articles, papers, documentation, tools, or open source repositories.
- `topics/` contains synthesized, durable notes about an area of cake or an idea that may feed several tasks or plans.
- `archived/` contains stale or superseded notes kept for historical reference.
